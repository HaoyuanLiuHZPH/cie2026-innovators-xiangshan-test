# 2026CIE全国RISC-V高水平创新及应用大赛（应用方向）

## 赛题名称
面向边缘AI大语言模型推理的RISC-V自定义vdot指令设计-基于“香山”昆明湖V2处理器架构

## 赛题背景与内容
- 点积（Dot Product）是深度学习模型的核心计算内核，在推荐系统、大语言模型（LLM）等AI应用中通常占据总计算量的60%–80%。在推荐系统中，78%的个性化推荐将点积作为核心排序指标，用于计算用户向量与商品向量的匹配度；在Transformer架构的LLM中，自注意力机制的矩阵乘法由海量点积构成，以GPT-2为例，点积占总计算量70%以上，其执行效率直接决定推理性能与能效，尤其在资源受限的边缘端更为关键。
- RISC-V凭借开源、模块化及可自定义扩展的优势，成为边缘AI处理器的主流架构。通过自定义向量点积指令（vdot），可在处理器流水线中集成专用硬件加速单元，以极小面积开销换取数倍点积吞吐提升，满足边缘LLM对高性能、低功耗、低成本的需求。“香山”昆明湖V2作为国产高性能RISC-V处理器，具备成熟的向量扩展与自定义指令接口，为vdot指令的硬件集成提供了理想的架构基础。
