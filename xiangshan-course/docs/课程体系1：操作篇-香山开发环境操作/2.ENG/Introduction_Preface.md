# Introduction: Preface

As you continue your studies, you will come to understand that XiangShan is an open-source, high-performance RISC-V processor project—not merely a single IP code repository, but a comprehensive engineering ecosystem that works in tandem and is widely used in the industry.

# What will you learn?
The “XiangShan Certification Curriculum” is designed to help learners develop technical competencies across **the entire XiangShan workflow, encompassing design, verification, software, and performance evaluation**.

# Why study “XiangShan” rather than other projects?
+ Currently the only open-source, high-performance RISC-V processor that is widely adopted by the industry as a commercial project
    - Open-source, big-core, and not a lab project
+ The “XiangShan” project not only provides code and documentation, but also a complete development environment and toolset, allowing you to actually get hands-on experience rather than just reading about it.
+ The skills gained from completing the “XiangShan” course can be applied to the development of any closed-source chip.

# Who is this course for?
This course is designed for developers with some familiarity with Linux, compilation, and engineering who are ready to get started with XiangShan in a practical setting.

We have repeatedly optimized the first chapter on the XiangShan development environment to ensure that over 95% of developers can successfully run “Hello, XiangShan.”



From an instructional perspective, this documentation **assumes the following skills **(though you can certainly learn as you go):

Familiarity with Linux and the shell

Understanding of the CPU verification process

Understanding of computer architecture

Familiarity with RISC-V

# Preparations Before Starting
If you are working on a local computer, we recommend the following:

+ Operating System: Windows 11
+ CPU: No specific requirements, but higher performance is better
+ RAM: 64 GB or more recommended
+ We recommend installing the package on an SSD
+ You will need to install the Linux Ubuntu operating system
