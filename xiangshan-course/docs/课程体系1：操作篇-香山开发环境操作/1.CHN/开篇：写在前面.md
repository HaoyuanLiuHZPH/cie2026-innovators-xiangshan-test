# 开篇：写在前面

通过接下来的学习，你将体会到：香山是一个开源高性能RISC-V处理器项目，不是单一的IP代码仓库，而是一整套协同工作的工程体系，被工业界广泛使用。

# 你将学到什么？
“香山认证课程体系”是学习**香山的设计 + 验证 + 软件 + 性能评估**一整条链路上所设计的技术能力。

# 为什么学习“香山”而不是其他项目？
+ 目前是唯一开源的RISC-V高性能处理器且被工业界广泛应用的市场化项目
    - 开源、大核、非实验室项目
+ “香山”项目不仅提供代码和文档，还提供了全套的开发环境及工具，让你可以真正上手操作而不是只看。
+ 完成“香山”相关课程学习所获得的能力，可迁移到任何一款闭源芯片的开发工作中

# 适合什么人学习？
本课程适合“懂一点 Linux / 编译 / 工程”的开发者上手学习香山实战。

第一章香山开发环境章节我们进行了反复优化，确保95%以上的开发者可以成功运行“Hello, XiangShan"



从教学视角看，文档**隐含假设了这些能力**：（当然你可以边看边学）

了解 Linux / shell

了解 CPU 验证流程

了解体系结构

听说过RISC-V

# 学习之前的准备工作
如果本地电脑进行操作，建议：

+ 操作系统：Windows 11
+ CPU：不做强制性要求，但是性能越高越好
+ 内存：推荐64G+
+ 整合包推荐放在固态硬盘中
+ 需要安装Linux Ubuntu操作系统
